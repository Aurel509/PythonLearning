1.1 Quelques informations de bases

Le langage de programmation Python a été créé en 1989 par Guido van Rossum, aux Pays-Bas. La première version publique de ce langage a été publiée en 1991.

Python est multiplateforme. C'est-à-dire qu'il fonctionne sur de nombreux systèmes d'exploitation : Windows, Mac OS X, Linux, Android, iOS...

C'est un langage de haut niveau. Il demande relativement peu de connaissance sur le fonctionnement d'un ordinateur pour être utilisé.

Python est un langage interprété, il n'a pas besoin d'être compilé pour être exécuté (contrairement au C/C++).

1.2 L'interpréteur

Comme dit précémment Python est un langage qui n'a pas besoin d'être compilé pour être éxecuté, possédant son propre interpréteur

Ce dernier peut être lancé sur un shell (cmd) avec la commande python

Le triple chevron >>> est l'invite de commande lorsque Python attend une commande de votre part. Testons cela avec une commande simple :

print("Salut !")

Voici le résultat que vous devez obtenir :

                     
    >>> print("Salut !")
    Salut !

Il également possible de s'en servir comme calculatrice, voici un exemple :

                     
    >>> 8+6
    14
    >>> 15*87
    1305

Vous pouvez quitter l'interpréteur avec la commande exit()

1.3 Premiers scripts et indentation

L'interpréteur peut s'avérer rapidement limité, d'où la possibilité de créer des scripts pouvant faire des choses un peu plus complexes.

Nous allons donc écrire notre premier script Python, les scripts en python possèdent l'extension .py. Créez votre script (nommez-le comme vous le souhaitez) avec cette extension quelque part sur votre machine. Vous pouvez utiliser un editeur de texte quelconque

Nous allons réutiliser la commande print() utilisée précédemment. 

                     
    #Mon premier script :D
    print("Je suis un message")

A noter, que vous pouvez ajouter des commentaires grâce au caractère #. Comme montré ci-dessus

Ensuite, enregistrez votre fichier, puis quittez l'éditeur de texte.

Pour exécuter votre script, ouvrez un shell et entrez la commande :

python monscript.py

Vous devrez donc retrouver ce résultat

                     
    $ python monscript.py
    Je suis un message

Les scripts que vous aurez à concevoir à l'avenir seront très probablement bien plus gros que ça. C'est pourquoi l'indentation est importante

En Python, l'espace blanc est utilisé pour indenter, le symbole deux points : pour le début d'un bloc, et le retrait pour la fin d'un bloc

Voici un exemple de structure : 

                    
    #Un bloc
    if truc == 52:
        print("truc est égal à 52")
    else:
        print("truc n'est pas égal à 52")

    #En dehors de tout les blocs. 
    print("du blabla inutile") 

    #Dans deuxième blc
    if autre_truc == True:
        if truc == 0:
            print("autre truc == true ! et truc est = 0 !")
        else:
            print("truc pas égal à 0")
            
        #Exécuté seulement si autre_truc == True
        valeur = truc
    else:
        print("autre truc pas égal a true")

A noter que les commentaires ne sont pas pris en compte dans l'indentation car ignorés par Python

En cas de non respect de l'indentation, le script affichera une erreur. Voici un exemple simple de mauvaise indentation

                     
    print("Ceci")
        print("N'est pas une bonne indentation")
    print("En effet.")
