6.1 - Définition et utilisation

Un tableau est une structure de données qui contient des valeurs.

En Python, un tableau peut contenir des types différents donnant une grande flexibilité à ces derniers. Ils sont déclarés par une série de valeurs séparées par des virgules et encadrées par des crochets

Les tableaux sont séparés en indices (ou index). 

                     
    tableau  : ['poubelle', 'carton', 'papier', 'les oignons']
    indice :            0         1         2              3

Notez bien que les tableaux commençent à 0 et non 1. Voici un exemple d'utilisation : 

                     
    >>> tableau = ['je','suis','un','tableau']
    >>> tableau[2]
    'un'

6.2 - Les fonctions avec les tableaux

Il existe différentes fonction qu'il est possible d'utiliser avec les tableaux.

La fonction len() permet de déterminer la longueur d'un tableau.

                     
    >>> tableau = ['carton','oignon']
    >>> len(tableau)
    2

Les fonctions de calculs min() max() sum() donnant respectivement le minimum, le maximum et la somme d'un tableau

                     
    >>> tableau = [11,5,9,8,4,0,1]
    >>> min(tableau)
    0
    >>> max(tableau)
    11
    >>> sum(tableau)
    38

La fonction append() permettant d'ajouter une valeur dans un tableau

                     
    >>> tableau = ['truc']
    >>> tableau.append('test')
    >>> tableau.append(123)
    >>> tableau
    ['truc', 'test', 123]

6.3 - Les tranches

Il est également possible de sélectionner une partie d'un tableau sur le modèle [m:n] allant de m à n

On parle donc de tranches, voici un exemple :

                     
    >>> tableau = ['je','suis','un','autre','tableau','sans','utilité']
    >>> tableau[0:3]
    ['je', 'suis', 'un']
    >>> tableau[2:5]
    ['un', 'autre', 'tableau']

Faites attention à ne pas dépasser les limites du tableau
