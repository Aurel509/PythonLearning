5.1 - Définition

Dans d'autres cas, vous allez être amené à devoir répeter des actions. C'est là que l'on retrouve les structures itératives aussi dites répétitives

Il existe deux cas essentiels de boucle :

  • La boucle for éxecute un code pour chaque élément d'un ensemble donné.
  • La boucle while éxécuter un code tant qu'une condition est vraie.

5.2 - Boucles for et while

Voici quelques exemples des structures itératives : 

                     
    valeur = 2

    #Tant que valeur pas égal à 15. On incrémente et affiche valeur 
    while(valeur != 15):
        valeur += 1
        print(valeur)
    
    valeur = 5

    #Pour i allant de 0 à valeur. On incrémente valeur de 5 et on l'affiche
    for i in range(valeur):
        valeur += 5
        print(valeur)
