2.1 Les bases

Python se démarque des autres langages quant à sa manière de gérer les variables, leur déclaration et initialisation se font en même temps.

On appelle ceci un typage dynamique, voici un exemple réalisé dans l'interpréteur :

                     
    >>> test = 10
    >>> test
    10

Dans cet exemple, nous déclarons une variable test, puis nous l'initialisation à 10

Mais en réalité, la ligne 1 permet ici de faire 3 choses à la fois

  • Premièrement, Python ici "devine" que notre variable est un entier
  • Deuxièment, Python alloue la mémoire nécéssaire pour stocker la variable et la retourner avec le nom test
  • Pour finir, Python assigne la valeur 10 à notre variable

2.2 Différents types

Comme vous le savez probablement, il existe plusieurs types de variables

  • Les entiers (integer) notés : int
  • Les nombres décimaux (ou réels) notés : float
  • Les chaînes de caractères notées : string
  • Les booléens (boolean) notés : bool
  • Et d'autres...

Voici d'autres exemples d'utilisation avec l'interpréteur : 

                     
    >>> a = 2.6
    >>> a
    2.6
    >>> b = "test"
    >>> b
    'test'
    >>> c = "Je suis une chaîne de caractères"
    >>> c
    'Je suis une chaîne de caractères'
    >>>

Dans cet exemple, on retrouve 3 variables, un float et 2 chaînes de caractères, les chaînes de caractères sont embriquées avec des guillemets et les floats utilisent un point en tant que séparateur décimal et non une virgule.

Ce qui veut dire que Python reconnait en tant que float 2.6 mais pas 2,6

Par ailleurs, vous pouvez consulter le type d'une variable avec la fonction type()

                     
    >>> e = 8
    >>> type(e)
    <class 'int'>

2.3 Nommage de variables

Le nom des variables en Python peut être constitué de lettres minuscules et majuscules, de nombres (0 à 9) ou d'un underscore _. Vous ne pouvez pas utiliser d'espace dans un nom de variable.

Python est sensible à la casse, les variables TesT, test ou TEsT sont donc différentes.

Certains noms de variables sont à éviter, notamment ceux "réservés" par Python.

Quelques exemples: if, print, from, range etc...

2.4 Opérations

On retrouve les quatre opérations de base sur les types numériques (entiers et floats) :

                     
    >>> valeur = 31
    >>> valeur + 9
    40
    >>> valeur - 3
    28
    >>> x = 1.85
    >>> x * 4
    7.4
    >>> y = 6
    >>> y / 2
    3.0

En fonction du calcul, la variable peut se transformer en float. Avec une division la transformation en float est systématique comme sur le dernier exemple.

Il existe également d'autres opérateurs.

  • % pour modulo
  • // pour le quotient
  • ** pour la puissance
                     
    >>> a = 6
    >>> 6 % 2
    0
    >>> a // 2
    3
    >>> a ** 2
    36

2.5 Conversions

A l'avenir il est possible que vous soyez amené à convertir des types. C'est-à-dire passer d'un type numérique à une chaîne de caractères ou vice-versa.

Heureusement, des fonctions ont été crées dans Python pour nous simplifier le travail.

A savoir:

int() float() et str()

                     
    >>> b = 12
    >>> str(b)
    '12'
    >>> c = 3.6
    >>> int(c)
    3
    >>> d = '42.1'
    >>> float(d)
    42.1
