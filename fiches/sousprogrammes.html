7.1 - Utilisation

En Python, afin de créer un sous programme (ou fonction) vous allez avoir besoin du mot-clé def et du mot-clé return dans le cas où vous souhaitez retourner quelque chose

Exemple de fonction avec retour

                     
    >>> def PlusUn(entier):
        return entier+1

    >>> PlusUn(8)
    9

7.2 - Les arguments

Lorsque l'on crée une fonction, il est possible d'avoir aucun ou plusieurs arguments. Lorsque le nombre d'arguments à l'appel de la fonction n'est pas respecté, une erreur apparait. 

                     
    >>> def GrosseAddition(e1,e2,e3,e4,e5):
	    return e1+e2+e3+e4+e5

    >>> GrosseAddition(1,2,4,8,1)
    16
    >>> GrosseAddition(7)
    Traceback (most recent call last):
    ...
    TypeError: GrosseAddition() missing 4 required positional arguments: 'e2', 'e3', 'e4', and 'e5'

7.3 - Les résultats de fonction

Les fonctions peuvent renvoyer plusieurs résultats à la fois.

En voici un exemple simple :

                     
    def PlusUnEtDeux(entier):
        return entier+1,entier+2

En réalité Python crée un tableau en tant que résultat, il est par conséquent possible de l'exploiter également. Comme pour récupérer uniquement la deuxième valeur par exemple 

                         
    def PlusUnEtDeux(entier):
        return entier+1,entier+2
            
    valeur = PlusUnEtDeux(7)
    print(valeur[1])
