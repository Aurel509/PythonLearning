4.1 - Bases et opérateurs de comparaisons

Très souvent, vous allez être amené à devoir comparer des élements entre eux afin d'éxecuter quelque chose de bien précis dans le cas où la comparaison est vérifiée. Par exemple écrire un texte uniquement si une valeur est égale à un nombre spécifique.

Il s'agit comme vous l'avez deviné de conditions

Les conditions sont construites autour de variables (selon leur valeur) d'où la nécéssité d'avoir des opérateurs afin de pouvoir les comparer.

En voici la liste :

  • == Tester une égalité entre deux élements
  • != Tester la différence entre deux éléments
  • > Tester si une valeur est strictement supérieure à une autre
  • < Tester si une valeur est strictement inférieure à une autre
  • <= Tester si une valeur est inférieure ou égale à une autre
  • >= Tester si une valeur est supérieure ou égale à une autre

4.2 - Conditions if et else

Python nous donne plusieurs structures conditionnelles :

  • La condition if (si)
  • La condition if else (si,sinon)
  • La condition if elif else (si, sinon si, sinon)

Voici quelques exemples de structures avec les différentes conditions :

                     
    valeur = 6
    valeur2 = 12

    #Condition si (if)

    if(valeur == valeur2):
        print("Valeur égales")

    #Condition si sinon (if else)
        
    if(valeur != valeur2):
        print("Valeur pas égales")
    else:
        print("Valeur égales")

    #Condition si, sinon si, sinon

    if(valeur > valeur2):
        print("Valeur supérieur")
    elif(valeur != valeur2):
        print("Valeur pas égales")
    else:
        print("Valeur inférieur et égale")
